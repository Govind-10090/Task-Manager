# Task Manager

## Description
A full CRUD task management application with secure authentication and a premium dark-themed UI. Users can sign up, log in, and manage their daily tasks with real-time updates.

## Tech Stack
- React
- Firebase (Authentication & Firestore)
- Vite

## Features
- User Authentication (Sign Up/Login)
- Create, Read, Update, Delete Tasks
- Real-time Data Synchronization
- Premium Glassmorphism UI
